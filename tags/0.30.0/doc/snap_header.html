gpsim - Snapshots

Occasionally Updated Snapshots

These snapshots are occasionally updated from the developmental code in the Subversion server. They are not official releases and should be considered "beta" quality. Some may not even work at all if a serious bug was introduced during development, so several are provided to give you the best chance of finding one that will work.

Because official gpsim releases are infrequent, these snapshots often contain fixes for bugs discovered in the last official release. If you do encounter a bug, please test the latest snapshot build to see if it has already been fixed by a developer.

To see what developers have added, you can view the ChangeLog directly from the Subversion server.
